HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык разметки веб-страниц во Всемирной паутине.
CSS (Cascading Style Sheets) — это код, который вы используете для стилизации вашей веб-страницы.
JavaScript — это язык программирования, который даёт возможность реализовывать сложное поведение веб-страницы.
